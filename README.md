# Basketball Mental Game Planner

An interactive web tool that helps basketball players build mental resilience and develop next-play mindset strategies.

## Features

- **Position-specific strategies** - Tailored for guards, wings, bigs, and all-rounders
- **Mistake-focused coaching** - Address specific mental challenges (missed shots, turnovers, foul trouble, etc.)
- **9 core mental strategies** including:
  - Reset routines
  - Next-play self-talk
  - Controllables focus
  - Mistake reframing
  - Body language awareness
  - Team connection
  - Post-game review
- **Auto-saving answers** - Your responses persist via localStorage
- **Downloadable game plan** - Export your personalized mental game plan as a text file

## Usage

1. Open `index.html` in any modern web browser
2. Select your position and the mistake that affects you most
3. Work through the strategy cards, answering the reflection questions
4. Use the "Save my plan" button to download your personalized mental game plan

## Development

This is a single-file HTML application with no build steps or dependencies. Just open and use.
