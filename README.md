# Basketball Mental Game Planner

An interactive web tool that helps basketball players build mental resilience and develop next-play mindset strategies, backed by sports psychology research.

## Features

### Position-Specific Experience
- **Dynamic color themes** - Each position has a unique color scheme:
  - Guard (Blue) - Court vision & orchestration
  - Wing (Emerald) - Versatility & balance
  - Big (Amber) - Power & presence
  - All-rounder (Purple) - Flexibility & adaptability
- **Personalized mental tips** - Research-backed advice tailored to your role

### Research-Backed Strategies
Strategies informed by leading sports psychologists:
- **Dr. Jim Loehr** - Ideal Performance State (IPS) & mental toughness training
- **Dr. Carol Dweck** - Growth mindset research
- **Dr. Bob Rotella** - Process focus & present-moment techniques
- **Dr. Patrick Cohn** - Quiet Eye technique for improved focus

### 9 Core Mental Strategies
1. Reset routines (3-5 second recovery)
2. Next-play self-talk
3. Controllables focus
4. Challenge reframing
5. "Misses are okay" rule
6. Team connection
7. Body language mastery
8. One-play focus targets
9. Post-game review (not replay)

### Interactive Features
- **Auto-saving answers** - Responses persist via localStorage
- **Downloadable game plan** - Export your personalized mental game plan
- **Random combo generator** - Practice different scenarios
- **Smooth theme transitions** - Visual feedback when switching positions
- **Progress indicators** - See which questions you've answered

### Accessibility
- Keyboard navigation support
- Skip-to-content link
- Focus-visible indicators
- Reduced motion support
- ARIA labels for screen readers

## Usage

1. Open `index.html` in any modern web browser
2. Select your position (theme changes automatically)
3. Select the challenge that tests you most
4. Read your position-specific mental tip
5. Work through the strategy cards, answering reflection questions
6. Use "Save my plan" to download your personalized mental game plan

## Development

Single-file HTML application with no build steps or dependencies.
